# 制作物紹介 #あのトンネル
終わりの見えないトンネルからの脱出を目指す、一人称3Dホラーゲームです。
![title](./title.png)
## 1．制作したきっかけ
小学生のころプレイしていたゲーム「妖怪ウォッチ2」内に登場するやりこみ要素「えんえんトンネル」を一人称視点にしたらさらに面白いと思い(本作は3人称後方視点)、さらにホラーゲーム全般が好きなため本作とは異なる要素で薄暗いトンネルにしてがっつりホラー演出を取り入れたいと思い制作した。
## 2．使用技術
- **Engine** : Unity 6000.2.7f2
- **言語** : C#
- **Render Pipeline** : URP(Universal Render Pipeline)
- **Platform** : PC(Windows)
